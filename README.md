# Open source API for latin_webstie

Built on Express + TypeScript + Prisma + PlanetScale.

## API Routes

-   /api/v1/auth
    -   [POST] /login
    -   [POST] /register
    -   [POST] /logout
    -   [POST] /users/search
    -   [GET] /users
    -   [GET] /users/me
    -   [DELETE] /users/:id
    -   [PATCH] /users/:id/roles
    -   [GET] /refresh_token
-   /api/v1/
    -   [POST] /books
    -   [GET] /books
    -   [POST] /books/:id/chapters
    -   [GET] /books/chapters/
-   /api/v1/
    -   [POST] /nouns
